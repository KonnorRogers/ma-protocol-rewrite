# About

This is a rewrite of the Massachusetts OEMS Protocols. This is not created by
Masschusetts OEMS, and is instead a different format of reading and searching the
protocols.

This rewrite is based off of the Official Version 2019.2

For the official Masschusetts OEMS Website [click here](https://www.mass.gov/orgs/office-of-emergency-medical-services)

For the official PDF [click here](https://www.mass.gov/doc/updated-emergency-medical-services-pre-hospital-treatment-protocols-version-20192-effective/download)
